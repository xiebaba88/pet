# 陪陪汪｜情侶共養小狗 (Peipei Pet) ✨

這是一個專為情侶設計的「雲端共養」數位寵物小遊戲。透過 Supabase 即時通訊技術，你們可以隨時隨地看到對方的操作，共同照顧這隻可愛的小狗。

## 🌟 重點功能

- **即時同步**：另一半正在摸狗狗、餵食或留言時，你都能立刻在畫面上看到愛心與特效。
- **PWA 支援**：可以安裝到手機桌面，像真正的 App 一樣開啟，不再受瀏覽器介面干擾。
- **成長紀錄**：包含留言板與心情日記，細心記錄你們與狗狗相處的每一天。
- **生動反應**：狗狗會根據心情、飽食度與體力產生不同的視覺表情與動作。
- **情侶限定**：專屬的「親密度」系統與相伴天數計算。
- **隱私門禁**：不硬編碼房間代碼，採本機記憶機制達成隱私安全。

## 🚀 如何在 GitHub Pages 上線？

1. **建立 Repo**：在你的 GitHub 帳號建立一個新的資料夾（例如叫 `pet`）。
2. **上傳檔案**：將 `index.html`, `manifest.json`, `sw.js` 這三個檔案上傳進去。
3. **開啟 Pages**：
   - 點擊 Repo 的 `Settings` -> `Pages`。
   - 在 `Build and deployment` 選擇 `Deploy from a branch`。
   - 分支選擇 `main` (或 `master`) 並點擊 `Save`。
4. **完成**：等待約 1 分鐘，你就會得到一個專屬的網址（如 `https://yourname.github.io/pet/`）。

## 📱 手機 App 化 (PWA)

1. 用手機瀏覽器打開你的網址。
2. **iOS (Safari)**：點選下方的「分享」按鈕 -> 選擇「加入主畫面」。
3. **Android (Chrome)**：點選右上方「三個點」 -> 選擇「安裝應用程式」或「加入主畫面」。

## ☁️ 關於 Supabase 與安全性

- **Supabase Key**：程式碼中的 `supabaseKey` 是 **Anon Key (匿名金鑰)**，設計上就是可以公開放在前端的。它配合後端的 **RLS (資料列層級安全性)** 運作，確保只有知道「房間代碼」的人才能讀取對應的資料。
- **防止流出 Service Role Key**：絕對不要將 Supabase 的 `service_role` 金鑰放在任何前端程式碼中。

## 🔐 房間門禁與本機記憶 (localStorage)

本專案使用 `localStorage` 技術來達成「免重複輸入」：
1. **第一次進入**：輸入你們專屬的房間代碼。
2. **本機儲存**：瀏覽器會將此代碼存在你手機/電腦的私密空間中（這不會傳給 GitHub 或伺服器）。
3. **自動進入**：下次開啟網頁時，程式會先檢查暫存，有的話就直接進入小屋。
